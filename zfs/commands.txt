Ось таблиця з усіма командами, про які я згадував раніше:

| Команда                                  | Опис                                               |
|------------------------------------------|----------------------------------------------------|
| `zpool create <pool_name> <device>`      | Створює новий пул ZFS з заданим іменем та пристроєм |
| `zpool destroy <pool_name>`              | Видаляє пул ZFS разом з усією його інформацією       |
| `zpool list`                             | Виводить список наявних пулів ZFS разом з їхнім станом, розміром та використанням |
| `zpool status`                           | Виводить стан пулу ZFS, включаючи стан кожного віртуального диску |
| `zpool add <pool_name> <device>`         | Додає новий пристрій до існуючого пулу ZFS           |
| `zpool remove <pool_name> <device>`      | Видаляє пристрій з пулу ZFS                          |
| `zfs create <pool_name>/<dataset_name>`  | Створює новий датасет ZFS у вказаному пулі           |
| `zfs destroy <pool_name>/<dataset_name>` | Видаляє датасет ZFS разом з усією його інформацією   |
| `zfs list`                               | Виводить список наявних датасетів ZFS                |
| `zfs snapshot <pool_name>/<dataset_name>@<snapshot_name>` | Створює знімок датасету ZFS                         |
| `zfs rollback <pool_name>/<dataset_name>@<snapshot_name>` | Відновлює датасет ZFS з вказаного знімку             |
| `zfs send <pool_name>/<dataset_name>@<snapshot_name>`     | Відправляє знімок датасету ZFS на інший сервер або в файл |
| `zfs receive <pool_name>/<dataset_name>`   | Приймає знімок датасету ZFS з іншого сервера або з файлу |

Ця таблиця містить список команд ZFS/Zpool, які можуть бути корисними при роботі з ZFS-файловою системою. Зверніть увагу, що перед використанням будь-якої команди рекомендується ознайомитися з документацією та вивчити деталі кожної команди для правильного використання їх у вашому середовищі.
