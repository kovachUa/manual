Звичайно! Ось загальна таблиця, яка включає всі параметри, що були пояснені:

| Параметр       | Опис                                                         |
| -------------- | ------------------------------------------------------------ |
| `driver`       | Вказує драйвер для волюму.                                    |
| `driver_opts`  | Налаштування, специфічні для обраного драйвера.               |
| `external`     | Вказує, що волюм знаходиться поза проектом Compose і вже створений. |
| `labels`       | Налаштування міток для волюму.                                |
| `name`         | Ім'я волюму.                                                 |
| `nocopy`       | Вказує, що необхідно використовувати існуючі дані волюму.      |

Ось оновлений приклад з вказаними параметрами:

```yaml
volumes:
  data_volume:
    driver: local
    driver_opts:
      type: nfs
    external: true
    labels:
      - "com.example.description=My volume"
      - "com.example.department=Finance"
  config_volume:
    name: my_config_volume
    nocopy: true
```

| Параметр        | Опис                                                         |
| --------------- | ------------------------------------------------------------ |
| `data_volume`   | Ім'я волюму `data_volume`, яке ви можете використовувати для посилання на цей волюм з інших сервісів або контейнерів. |
| `driver`        | Драйвер волюму для `data_volume`. В цьому випадку використовується значення `local`. |
| `driver_opts`   | Додаткові налаштування для драйвера волюму. В цьому випадку використовується `type: nfs` для вказання типу файлової системи NFS. |
| `external`      | Вказує, що `data_volume` знаходиться поза проектом Compose і вже створений. |
| `labels`        | Мітки, які призначені волюму `data_volume`. У цьому випадку, мітки `com.example.description=My volume` та `com.example.department=Finance` використовуються для опису та класифікації волюму. |
| `config_volume` | Ім'я волюму `config_volume`, яке ви можете використовувати для посилання на цей волюм з інших сервісів або контейнерів. |
| `name`          | Ім'я волюму `config_volume` задане як `my_config_volume

`.      |
| `nocopy`        | Значення `true` вказує, що не потрібно копіювати дані у волюм з контейнера. |

Ця таблиця містить усю інформацію, яку я пояснював раніше, зібрану в одному місці для вашої зручності.
